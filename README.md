# Překladač

## Návrh jazyka

Jazyk kalkulačky

#### Datové typy
- int 32
- float 32
- boolean 1 (true / false)
#### Deklarace proměnných
- int x
- float y
- boolean z

#### Operátory

- +, -, *, /, (, ), [, ]

#### Funkce

- absolutní hodnota: abs(a)
- mocnina: pow(a, n)
- odmocnina: sqrt(a)
- rozklad na prvočísla: fact(a)

#### Správný výraz

((a+a-b)*a/b) + abs(a-b) - pow(a, 3)
